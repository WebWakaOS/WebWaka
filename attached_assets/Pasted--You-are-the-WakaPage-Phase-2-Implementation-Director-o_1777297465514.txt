### Stage A — Renderer Foundation (apps/brand-runtime)
1. **Create `apps/brand-runtime/src/routes/wakapage.ts`** — Main public renderer route
   ```
   GET /{tenant-slug}/wakapage → render published page + blocks
   GET /{tenant-slug}/wakapage/{page-slug} → render specific page
   ```

   ```
   1. custom_domain → tenant_id (tenant_branding)

   4. 404 if no tenant or no published WakaPage
   ```
