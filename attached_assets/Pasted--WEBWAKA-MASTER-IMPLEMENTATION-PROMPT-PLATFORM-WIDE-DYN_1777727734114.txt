- override logic

LAYER 3 — DYNAMIC AUTHORIZATION + PRIVILEGE SYSTEM
Replace static privilege assumptions with runtime-managed access control combining:
- base system roles
- custom roles

- permission bundles
- scoped permissions

Implement a hierarchical control model:

- Sensitive operations can have approval or restriction rules
- All changes are audit-logged

- emergency kill switches
- enable/disable of major features
